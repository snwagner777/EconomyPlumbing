# ServiceTitan API - Delete Location Contact

## Endpoint
**DELETE** `https://api.servicetitan.io/crm/v2/tenant/{tenant}/locations/{id}/contacts/{contactId}`

## Request Parameters
- `id` (template, required, integer) - Location ID
- `contactId` (template, required, integer) - Contact ID
- `tenant` (template, required, integer) - Tenant ID

## Response: 200 OK
The request has succeeded
